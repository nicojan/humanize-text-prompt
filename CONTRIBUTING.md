# Contributing

Thanks for your interest in improving this prompt. Here's how to help.

## Reporting a new research paper

If you've found a peer-reviewed paper on AI text detection that identifies patterns not covered by the current rules:

1. Open an issue using the "New research" template.
2. Include the full APA citation.
3. Summarize the relevant findings (what pattern was found, in what data, by what method).
4. Note which layer of the prompt it would affect (vocabulary, structure, tone, discourse, texture).

## Proposing a new rule

If you've identified an AI writing pattern through your own editing experience:

1. Open an issue using the "New pattern" template or submit a PR directly.
2. Include a before/after example showing the pattern and its fix.
3. Cite research support if available. Experiential observations are welcome but should be flagged as such.
4. Specify which section of `PROMPT.md` the rule belongs in.

## Editing existing rules

PRs that improve clarity, fix errors, or add examples to existing rules are welcome. Please keep the tone consistent with the rest of the document: direct, specific, and free of the AI patterns the prompt is designed to eliminate.

## Style guidelines for contributions

- Write in the same voice as the prompt: concise, opinionated, concrete.
- Cite sources in APA 7th edition format.
- Every rule should have an actionable "do this" component, not just a "this is bad" observation.
- Include a before/after example for any new anti-pattern.
- Don't add rules that only apply to one specific LLM unless noted as model-specific.

## What not to contribute

- AI detection tools or classifiers. This is an editing guide, not a detector.
- Rules that contradict existing rules without explanation.
- Marketing language or promotional content.
- Rules without any research or experiential basis.
